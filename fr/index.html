Introduction

Bienvenue dans PolitiScales, le test politique qui vous situe politiquement. Vous allez être confronté à une série d'affirmations. Pour chacune d'elles, cliquez sur le bouton correspondant à votre opinion.


Les questions supposent que vous êtes citoyen d’une nation avec un système politique multi-parti et une économie de marché. Mais si cela n’est pas le cas, ce test peut être quand même fiable pour une très grande majorité des réponses.

Si vous ne comprenez pas le sens d'une question, essayez de faire des recherches sur internet pour mieux comprendre son sens. Ne mettez pas une réponse au hasard pour ne pas fausser le résultat.

Ce test cherche à representer le panel le plus large possible d'opinions et contient donc des phrases qui peuvent vous choquer, en particulier concernant le racisme et l'homophobie.
